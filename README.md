## Sections

1. **Hero Section** - Introduction with call-to-action buttons
2. **About** - Personal information and statistics
3. **Skills** - Technical skills organized by category
4. **Projects** - Featured projects with descriptions
5. **Contact** - Contact form and social links

## Customization

### Personal Information

1. **Name and Title**: Update in `index.html`:
   - Line 30: Change "Your Name"
   - Line 31: Update the subtitle
   - Line 32-34: Modify the description

2. **About Section**: Edit the about text in the About section (around line 70)

3. **Contact Information**: Update contact details in the Contact section:
   - Email address
   - Phone number
   - Location

4. **Social Links**: Update the social media links in the Contact section (GitHub, LinkedIn, Twitter)

### Skills

Edit the skills in the Skills section. Each category can be modified or new categories can be added.

### Projects

Replace the placeholder projects with your actual projects:
- Update project titles
- Add project descriptions
- Modify technology tags
- Add links to live demos and GitHub repositories
- Replace placeholder images with actual project screenshots

### Colors

The color scheme can be customized in `styles.css` by modifying the CSS variables at the top:

```css
:root {
    --primary-color: #6366f1;
    --primary-dark: #4f46e5;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

## Getting Started

1. Open `index.html` in a web browser
2. That's it! No build process required.

## Deployment

You can deploy this portfolio to:
- **GitHub Pages**: Push to a GitHub repository and enable Pages
- **Netlify**: Drag and drop the folder or connect your Git repository
- **Vercel**: Import your Git repository
- **Any static hosting service**
